You have to write a piece of code, a function "combine(list, number)" which is given:

a list of the available resistors and the requested value and returns a string describing the combination searched for.

The returned string shall be made up of functions describing how all is connected. The combination shall be made entirely of parallel and series connections (for simplicity so called "bridged circuitry" is not needed).

These two functions to do so are: 

"series(*a)" for series connections where "*a" is R1, R2, R3, ....      
"parallel(*a)" for parallel connections where "*a" is R1, R2, R3, ....
(A single resistor can be represented by its value alone.)

A few examples : 

combine([3, 3, 6], 5) == "series(3, parallel(3, 6))"

combine([20, 20, 30, 30], 24) == "series(parallel(20, 30), parallel(20, 30))"
                              or "parallel(series(20, 20), series(30, 30))"

combine([12, 12, 12, 12], 3) == "parallel(12, 12, 12, 12)"
                             or "parallel(parallel(12, 12), parallel(12, 12))"

combine([24, 12], 12) == "12"   # (one single resistor)

combine([6, 9], 12) == "None"   # (Sorry, can't be done)

The on the mission involved mathematics :

You can calculate the total resistance of two resistors (R1, R2) series connected (Rs) by: 

Rs = R1 + R2

And the calculation for two resistors (R1, R2) parallel connected (Rp) is: 

Rp = 1 / (1 / R1 + 1 / R2)

Which can albo be written as : 

Rp = R1 * R2 / (R1 + R2)
